# 冰激凌球

极圈线南移的那一年，主角十二岁。她叫什么不重要，重要的可能是她出生在在北纬三十一度的某个城市，那里曾经以梅花著称。先是哈尔滨的地铁冻裂了管道，接着是莫斯科全城停电，再然后新闻里开始出现一些听不懂的词——急冻锋面、零下阈值、冰晶肺。大人们关掉电视，说一切都会好的。

太阳没有熄灭，这是比较搞笑的一点：它还在那里，每天照常升起，在铁灰色的天空里磨出一块惨白的光斑，然后沉下去。但它现在很像冷链柜的照明灯。梅花则早就没有了。

一切也没有好。温度每天都跌一点，像一只看不见的手把全世界的恒温器向左拧到底。赤道的最后一茬水稻在巴西收完，粮食带便彻底消失了。各国政府在还能运转的时候做了一件事：他们选了人，送上近地轨道。

应该有很多是科学家、工程师，即那些在电影里会被拯救的人吧。但是他们显然不是重点，或者说他们其实不是居民而是物业。居民是账户余额最长的那一批。圣徒，他们这么称呼自己。很难确认他们有多少人，但应该全部住在近地轨道的环形舱里，窗外是充沛的日照——没有大气层的衰减，阳光纯得像刚铸出来的金子。他们在上面晒日光浴，透过穹顶看地球一点一点变成白色。听说还办了泳池派对。

而留在地表的人，起初应该是六十亿以上，后来就不知道了。有人觉得未必会比天上更多。

天上，或者说“天堂”，不是一个单一的宫殿，而是某种群岛，一个由无数天使构成的国：

所有轨道飞行器统称"天使"。不同层级对应不同的职能与等级。层级名称似乎是古典天使学九阶体系。对于每一级的船只，型号代际不一，是在多年间陆续建成的。

地面上可能没有人恨他们。恨需要力气，而力气很适合用来生火。晚上能看见空中飘过这人造的“星座”，像萤火虫。

城市被冻成了雕塑。上海的玻璃幕墙在零下九十度全部崩裂，碎片悬在半空的那一刻就被冻在原地，远看像一串静止的流星。东京塔的铁骨脆得像饼干，一场夜风之后只剩基座。纽约的地铁隧道成了天然冷库，里面坐着很多人，像冷链运输的金枪鱼。

活着成了一种惯性。人们每天醒来检查自己还有几根指头，把能找到的有机物塞进嘴里，然后等天黑。世界安静得可怕，没有鸟叫，没有汽车，没有人的声音。偶尔能听到冰层裂开的巨响，那说明又一座建筑塌了。

但也有奇怪的美。日落的时候，冰原会反射出粉紫色的光，像整个地球都在燃烧——只不过烧的是冷焰。极光蔓延到了曾被称为热带的纬度，绿色的光幕在头顶流动，像某个还活着的东西在呼吸。

主角这些被留下来的人渐渐分了群。有人往南走，相信赤道附近还有余温。有人把自己埋在冻土里，说可以像熊一样冬眠。有人不停地发电报，朝轨道舱的方向——从来没有收到过回复。

无线电有时能收到不少颇有希望的消息，或者至少那些人是这么认为的。有人说他们找到了解决方案，有人说要发起什么工程，还有一些搞不好是从轨道上发来的，但所有都是破碎的只言片语。

主角哪也没去。她似乎在等春天。
